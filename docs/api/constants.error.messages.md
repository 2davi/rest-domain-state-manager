# constants/error.messages

에러 및 경고 메시지 상수

라이브러리 전체에서 발생하는 모든 Error/Warning 메시지를 중앙 관리한다.
메시지 함수는 컨텍스트 값을 인자로 받아 최종 문자열을 반환한다.
