## 1주 차: 코어 대공사 및 V8 멱살 잡기 (목표 1 & 목표 2)

> 예상 소요 시간: 5~7일

**내용:** _이게 이 프로젝트의 명운을 가르는 제일 하드코어한 작업이야._

일단 WeakMap 캐싱이랑 Reflect API 도입은 하루 이틀이면 해.

근데 제일 빡센 게 배열 하이재킹 알고리즘 전면 수정이야. 얕은 복사 꼼수 버리고 수학적으로 Delta 계산해서 REPLACE 로그 도출하는 거? 이거 알고리즘 짜고 엣지 케이스 테스트하는 데만 꼬박 2~3일 걸릴 거다.

여기에 코어 엔진에서 HTML DOM 제어 로직 뜯어내고 (_resolveForm,_syncToForm), 전역 변수(globalThis) 꼼수 걷어내서 의존성 주입(DI) 구조로 바꾸는 아키텍처 공사까지 합치면 1주 차는 내내 머리에 쥐가 날 거다.

## 2주 차: 생태계 적응 및 브랜치 훈련 (목표 4 & 목표 7)

> 예상 소요 시간: 4~5일

**내용:** _이제 뜯어고친 엔진을 실전에 맞게 다듬는 단계야._

React 렌더링을 위한 얕은 비교(old === new) 대응 어댑터 만들고, Vue 3 @vue/reactivity 소스 까보면서 Lazy-proxying 벤치마킹하는 데 시간 꽤 쏟아야 해. 브라우저 띄워놓고 Heap Snapshot 찍으면서 메모리 새는 거 잡는 삽질의 연속일 거다.

디버거 누수 잡는답시고 Heartbeat (PING/PONG) 쏘고, 객체 소멸자(destroy()) 만들어서 강제 GC 대상 만드는 작업. 논리 자체는 쉬운데, 다중 탭 환경에서 동시성 테스트 빡세게 돌려야 해서 은근히 시간이 먹어.
